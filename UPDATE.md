# v1.1 (beta)
Изменения:
- Добавлена команда `.removewl` для удаления пользователя из белого списка
- Изменена система хранения белого списка, вместо `whitelist.txt` будет использоваться `whitelist.json` в стабильном формате, также исправлена проблема дублирования
- Добавлена система конфигурации (`config.json`) для изменения любых системных промптов, сообщений и параметров
